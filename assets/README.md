# Assets de LudopatIA

Estructura sugerida:

- assets/fonts/ : fuentes descargadas (.ttf, .otf, .woff2)
- assets/sprites/ : sprites de personajes, iconos, fondos en pixel art
- assets/audio/ : musica y efectos de sonido
- assets/ui/ : elementos de interfaz (marcos, botones, logos)

Sugerencia de nombres para integracion rapida:

- assets/fonts/pixel-main.ttf
- assets/sprites/title-logo.png
- assets/sprites/ia-character.png
- assets/audio/menu-theme.mp3
